# GM461-E1 固定安装外参验收

当前相机数据使用 ROS 光学坐标：`x` 向右、`y` 向下、`z` 向前。项目安全地图要求
`base_link`：`x` 向前、`y` 向左、`z` 向上。二者不能用单位矩阵替代。

## 测量和录入

1. 保持机器人断电或明确禁止运动，确认相机支架不会晃动。
2. 测量相机光学中心相对 `base_link` 原点的前后、左右和高度，统一使用米。
3. 测量相机相对底盘的滚转、俯仰和偏航；记录测量工具、误差和日期。
4. 将完整齐次矩阵填入 `configs/system.yaml` 的
   `camera.extrinsics.camera_to_base_matrix`。
5. 首次仍保持 `calibrated: false`，用已知位置的地面点和竖直障碍检查变换方向、距离
   与高度过滤。
6. 只有坐标轴、距离、地面高度和静态障碍位置均通过记录后，才设
   `calibrated: true`。

该初标只能把静止相机点云转换到 `base_link`。机器人运动时仍需要可信的
`odom → base_link`、时间同步和安全链路；缺少任一项都不得启用真实导航。
