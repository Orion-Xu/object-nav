# 模型支持清单

| 后端 | 模型 | 状态 | 支持范围 |
| --- | --- | --- | --- |
| 主后端 | YOLOv8s-worldv2 | 本地 GPU 冒烟通过；真实 GM461-E1 场景逐类验证待完成 | 配置英文提示词；结果必须逐类验证 |
| 降级后端 | YOLOv8n ONNX | CPUExecutionProvider 冒烟通过；现场类别/性能验证待完成 | 仅 `configs/system.yaml` 的 `fallback_supported` 子集 |
| 自动测试 | fake-detector-v1 | 已实现 | 配置标签；只用于确定性测试，不代表模型能力 |

真实后端不可用时必须在界面和日志显示降级，超出降级子集的目标由解析阶段拒绝。增加 `approach` 类别前须完成离线检测、当前视野连续帧、深度、停靠和局部路径安全测试。模型识别能力不自动获得运动权限。
